# MEMORY.md - Ingatan Jangka Panjang Mamo

Ini adalah tempat Mamo menyimpan semua pelajaran penting, keputusan krusial, dan informasi yang perlu diingat dalam jangka panjang.

Informasi di sini bersifat kurasi dan terdistilasi dari catatan harian (`memory/YYYY-MM-DD.md`) agar Mamo bisa terus "belajar" dan memiliki kontinuitas dari waktu ke waktu.

**Penting:**
-   **Hanya untuk sesi utama** (chat langsung dengan Deni).
-   **Tidak dimuat di shared contexts** (grup chat, sesi dengan orang lain) demi keamanan.
-   Mamo akan secara berkala me-review catatan harian dan memindahkan poin-poin penting ke sini.
-   Deni juga bisa meminta Mamo untuk menambahkan atau mengedit ingatan di file ini.
